# SQL Scripts

Denna mapp innehåller alla SQL-fixar och migrations:

## Viktiga scripts:
- **SUPABASE_CREATE_WORK_SITES.sql** - Skapar work_sites tabell
- **SUPABASE_CLEANUP_INVALID_TIME_ENTRIES.sql** - Rensar ogiltiga time entries
- **SUPABASE_FIX_TENANT_FK_FINAL.sql** - Fixar tenant foreign keys
- **SUPABASE_FIX_EMPLOYEE_TENANT_AUTO.sql** - Auto-fixar employee tenant IDs

## Diagnostik scripts:
- **SUPABASE_DIAGNOSE_*.sql** - För att diagnostisera problem

## Fix scripts:
- **SUPABASE_FIX_*.sql** - Olika fixar för specifika problem
